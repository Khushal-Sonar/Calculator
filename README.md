# Simple Python Expression Calculator

A lightweight, terminal-based Python calculator that can evaluate full mathematical expressions with multiple numbers at once.

## ✨ Features
* **Multi-Number Operations:** Enter long expressions like `5 + 6 + 7 - 6` in a single line.
* **Standard Order of Operations:** Automatically follows standard math rules (multiplication and division before addition and subtraction).
* **Input Validation:** Uses basic security checks to ensure only numbers and math operators are processed.
* **Crash-Proof:** Gracefully handles invalid inputs and prevents crashes from dividing by zero.

## 🚀 How to Run
1. Make sure you have Python installed on your machine.
2. Save the script as `calculator.py`.
